# Bíblia de Motion Design — MilhasPix

Guia visual completo de produção de vídeos para a MilhasPix.

## Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**

## Como rodar localmente

```bash
npm install
npm run dev
```

## Deploy

Hosteado no Vercel. Todo push na `main` faz deploy automático.
